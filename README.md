[Mangu MVC Framework](http://madradavid.com) 
=================

Mangu is an easy to use PHP MVC Framework.



Quick start
-----------

Clone the repo, `https://github.com/madra/mangu.git`, or [download the latest release](https://github.com/madra/mangu/downloads).
